⚙ How to add this card to presentations.html

  1. 1 Upload your PNG thumbnail to the images/ folder in the csg-guides GitHub repository.
  2. 2 Open presentations.html in the repository and click the pencil icon (Edit this file) in the top-right corner.
  3. 3 Use Ctrl + F (or Cmd + F on Mac) to search for the section comment, e.g. <!-- CYBER SAFETY GUY PRESENTATIONS -->, to find the right place.
  4. 4 Scroll to the last </article> tag before the closing </div> of the pres-grid section.
  5. 5 Paste your generated card code directly after that closing </article> tag.
  6. 6 Scroll to the bottom of the page, add a short commit message (e.g. Add new presentation card), and click Commit changes. Allow up to 2 minutes for GitHub Pages to update.
