Where to paste this

  1. Open index.html in your GitHub repository
  2. Click the pencil (edit) icon to edit the file
  3. Use Ctrl+F to search for the category comment, e.g.
  4. Find the last </a> tag before the closing </div> of that category's guide-grid
  5. Paste your new card code immediately after that </a> tag
  6. Scroll down and click Commit changes
  7. Wait 2 minutes then hard refresh your page — your new guide will be live
